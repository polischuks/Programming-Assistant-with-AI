Description

In this stage, the chatbot becomes even more personalized, using information provided by the user in the previous stage. The bot will not only greet the user by name but also adapt its questions and recommendations based on the user's interests and level of knowledge in programming. Integrating the ChatGPT API will allow the bot to offer deeper and more personalized advice, guides, and resources for learning programming.

Objectives

The program should take into account the information obtained in the previous stage to create a deeper and more personal interaction with the user. The bot should:

Greet the user by name.
Ask more in-depth questions based on the user's previous responses.
Use the ChatGPT API to generate individualized advice and recommendations.

Task

Expand the bot's functionality using the information obtained from the user in the first stage.
Integrate the ChatGPT API to tailor the dialogue to the user's interests and knowledge level.
Offer the user personalized learning materials based on their interests.
