Description:

In this initial phase of the project, you will develop a prototype chatbot. This prototype will simulate interactions by providing scripted responses based on the time of day and the user's input regarding their interests in programming. The primary goal at this stage is to draft a user interaction script that showcases how the chatbot would behave in various scenarios. It is important to note that this version of the chatbot will not incorporate smart logic, external API calls, or real-time interaction with the ChatGPT API. The focus is on laying the groundwork for a simple yet interactive chatbot capable of introducing itself, adapting its greeting according to the time of day, and engaging the user with questions about their programming interests. Based on the user's responses, the chatbot will offer basic resources or advice tailored to the user's interests. This mockup serves as a foundational step, enabling us to visualize and refine the chatbot's interaction flow before integrating more complex functionalities in future iterations.

Objectives:

  • Draft a Script: Write a detailed script that outlines the chatbot's interactions, including how it adapts its greetings based on the time of day and how it inquires about the user's programming interests.

  • Simulate User Interaction: Create a mockup that simulates the chatbot's behavior in response to predefined user inputs. This should include a variety of greeting messages corresponding to different times of the day and tailored responses based on the user's stated programming interests.

  • Provide Personalized Recommendations: Use the simulated input from the user to generate scripted recommendations or advice related to programming resources. This aspect of the mockup should demonstrate how the chatbot could potentially offer value to the user based on their specific interests.

Example 1: Morning Greeting and Interest in Web Development

  • User Input: "Hello"
  • Bot Output: "Good morning! My name is Bot, and I'm here to help you with programming. What area of programming are you interested in?"
  • User Input: "I'm interested in web development."
  • Bot Output: "Great choice! Web development is an exciting field. I recommend starting with learning HTML, CSS, and JavaScript. There's an excellent resource for beginners at hyperskill.org."

Example 2: Afternoon Greeting and Interest in Artificial Intelligence

  • User Input: "Hello"
  • Bot Output: "Good afternoon! I'm Bot, and my task is to assist you in learning programming. What areas are you interested in?"
  • User Input: "Artificial intelligence."
  • Bot Output: "Artificial intelligence opens up many possibilities. I suggest learning Python and libraries like TensorFlow or PyTorch. You can start with courses on hyperskill.org."

Example 3: Evening Greeting and Interest in Mobile Development

  • User Input: "Good evening"
  • Bot Output: "Good evening! I'm Bot, your programming assistant. What area would you like to improve your skills in?"
  • User Input: "Mobile development."
  • Bot Output: "Mobile development is very popular right now. To begin, I recommend getting familiar with Swift for iOS or Kotlin for Android. You can find an excellent introductory course on hyperskill.org."

Example 4: Night Greeting and Interest in Databases

  • User Input: "Hello"
  • Bot Output: "Good night! I'm here to help you with programming. What area of programming interests you the most?"
  • User Input: "Databases."
  • Bot Output: "Databases are the cornerstone of many applications. I suggest starting with SQL, and then moving on to NoSQL databases. You can find a good educational course on hyperskill.org."
