Description

In this stage, the chatbot becomes even more interactive by learning to count out loud up to a number chosen by the user. Integrating the ChatGPT API will allow the bot to not only perform the count but also make it interactive, for example, by commenting on certain numbers or challenging the user to guess which numbers the bot will find special. This will add a playful element to the learning process and make interacting with the bot more engaging.

Objectives

The objective of this stage is to develop the bot's ability to count numbers in an interactive format, using the ChatGPT API to generate interesting facts or tasks related to the numbers. The bot should:

Ask the user for a number to count to.
Count from 0 to the specified number, offering interactive comments or tasks at certain numbers.
Use the ChatGPT API to create these interactive moments.

Task

Implement a function that allows the bot to count from 0 to the user-specified number.
Integrate the ChatGPT API to generate interesting comments or questions about the numbers during the count.
Offer the user an interactive task related to numbers, such as guessing at which number a special comment will be made.
