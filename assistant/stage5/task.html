Description

In this final stage, the chatbot will engage the user in an interactive programming quiz. The quiz will consist of multiple-choice questions related to programming concepts and languages. Utilizing the ChatGPT API, the bot will dynamically generate questions, options, and feedback tailored to the user's responses. Upon answering each question, the bot will assess the user's input, provide immediate feedback, and offer relevant hints or educational resources for further learning. Correct answers will be acknowledged with positive reinforcement, while incorrect ones will prompt the bot to offer guidance and supplementary materials.

Objectives

The primary objective of this stage is to enhance user engagement and learning through interactive quiz-based interaction. The chatbot should:

  • Generate programming-related quiz questions and multiple-choice options.
  • Evaluate user responses and deliver appropriate feedback.
  • Dynamically utilize the ChatGPT API to generate hints, explanations, and additional learning resources based on user interactions.

Task

Develop a quiz system featuring programming-related questions and multiple-choice answers. Integrate the ChatGPT API to generate dynamic quiz content and feedback. Provide users with educational resources, hints, and explanations to aid learning and reinforce understanding of programming concepts.
